# BabyLog v0.1 — MVP 基础功能（已完成）

## 目标

搭建一个**可用的私有家庭宝宝记录系统**，验证核心流程。

## 范围

- 一个 family token 对应一个家庭，可记录多个宝宝
- 7 种事件类型：feed、poop、weight、vaccine、sleep、diaper、solid
- 移动端：注册/登录（token）、快速记录、时间线、设置（FaceID、添加宝宝、退出）
- API：REST + Bearer token；babies 和 events 的 CRUD；stats stub 接口
- E2E 测试：Playwright（Web）+ Maestro（iOS）

## 不在范围

- 多租户/公开发现
- 实时推送
- 真实统计（v0.1 为 stub）
- Web 端（mobile-first）
