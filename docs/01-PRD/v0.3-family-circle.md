# BabyLog v0.3 — 家庭圈

## 目标

让宝宝的成长不再只有爸爸妈妈知道——邀请爷爷奶奶、外公外婆、亲友加入家庭，一起查看宝宝的照片、里程碑和日常记录。同时补全数据洞察能力，让喂养和成长数据真正有用。

## 用户故事

> 作为一个新手妈妈，我希望：
> - 把宝宝的第一次走路照片分享给远在老家的奶奶看（家庭共享）
> - 奶奶只需要一个邀请码就能加入，不用学复杂操作（简单邀请）
> - 奶奶能看但不能误删我的记录（角色权限）
> - 保姆阿姨能帮我记录白天的喂奶和换尿布（caregiver 角色）
> - 带宝宝看医生时给医生看喂奶和体重趋势（统计图表 + 导出）

## 功能模块

### 模块一：家庭成员与角色系统

#### 1.1 数据模型

```
families (保持)
  |
  +-- family_members (新表)
  |     id, family_id, name, role, token_hash, invited_by, joined_at
  |     role: 'admin' | 'caregiver' | 'viewer'
  |
  +-- family_invites (新表)
  |     id, family_id, code, role, name, created_by, expires_at, used_at
  |
  +-- babies (保持)
  +-- events (增加 recorded_by → family_members.id)
```

#### 1.2 用户资料迁移

v0.2 中的本地用户称呼和头像需迁移到服务端：
- 首次加入 family_members 时，自动将本地称呼和头像上传到服务端
- 后续修改称呼/头像直接走 API
- 清理本地 AsyncStorage 中的旧数据

#### 1.3 三种角色

- **admin（爸爸/妈妈）**：全部权限——记录、编辑、删除、管理宝宝、管理成员、邀请
- **caregiver（看护人/保姆）**：记录事件 + 查看所有数据，不能删除数据或管理成员
- **viewer（爷爷奶奶/外公外婆）**：只能查看时间线、照片、里程碑、统计

#### 1.3 权限矩阵

| 操作 | admin | caregiver | viewer |
|------|-------|-----------|--------|
| 查看时间线/照片 | Y | Y | Y |
| 添加事件/记录 | Y | Y | N |
| 编辑/删除事件 | Y | 仅自己记录 | N |
| 管理宝宝信息 | Y | N | N |
| 邀请成员 | Y | N | N |
| 管理/移除成员 | Y | N | N |
| 导出数据 | Y | N | N |
| 修改自己的称呼 | Y | Y | Y |

### 模块二：邀请流程

1. Admin 在设置页 → "家庭成员" → "邀请家人"
2. 选择角色 + 输入称呼（如"奶奶"、"外公"、"李阿姨"）
3. 生成 6 位邀请码（有效期 7 天）+ 可复制的邀请文案
4. 通过微信/短信/复制分享给家人
5. 家人下载 app → 欢迎页选"加入家庭" → 输入邀请码 → 设置自己的称呼 → 获得独立 token
6. 邀请码使用后自动失效（一码一用）

**API：**
- `POST /invites` — 创建邀请（admin only）
- `POST /invites/:code/accept` — 接受邀请（无需 auth）
- `GET /invites` — 查看未使用的邀请列表（admin only）
- `DELETE /invites/:id` — 撤销邀请（admin only）

### 模块三：家庭成员管理

- 设置页"家庭成员"列表：显示成员称呼、角色、加入时间
- Admin 可：修改成员角色、移除成员
- 所有成员可：修改自己的称呼
- 移除成员后其 token 立即失效

**API：**
- `GET /members` — 成员列表
- `PATCH /members/:id` — 修改角色/称呼
- `DELETE /members/:id` — 移除成员（admin only）

### 模块四：家庭共享时间线

- 所有角色登录后看到同一个家庭时间线
- 事件卡片显示记录人标签（"妈妈 · 3/29 10:30"）
- Viewer 看不到编辑/删除按钮
- Caregiver 只能编辑/删除自己记录的事件
- 里程碑、照片对所有成员可见

### 模块五：数据与洞察

#### 5.1 统计图表
- 喂奶趋势：近 7 天每天喂奶次数 + 总量折线图
- 睡眠模式：近 7 天睡眠时长柱状图 + 时段分布
- 体重曲线：随时间变化 + WHO 百分位参考线
- 替换 v0.1 的 stub API 为真实查询
- 新增统计页面入口

#### 5.2 数据导出
- 设置页"导出数据"入口（admin only）
- PDF 报告：宝宝基本信息 + 日期范围事件汇总 + 图表
- 支持系统分享

## 技术变更摘要

- `services/api/migrations`：新增 `family_members` 表、`family_invites` 表；events 增加 `recorded_by`
- `services/api/src`：角色权限校验中间件；邀请相关路由；成员管理路由；统计查询替换 stub
- `apps/mobile`：新增"加入家庭"入口（欢迎页）；成员管理页面；邀请流程 UI；统计图表页面
- Auth 机制扩展：token 关联到 family_member 而非 family

## 不在本期范围

- 用户账号体系升级（→ v1.0）
- 视频上传（→ v1.0）
- 实时推送通知
