# BabyLog 移动端 UI 风格指南

本文档描述 BabyLog App 的视觉与交互风格，便于后续开发保持统一。风格参考：简洁白底 + 薰衣草紫主色 + 圆角卡片式布局（类似 “Welcome back to Jour” 登录页）。

---

## 1. 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#FFFFFF` | 主背景，保持干净 |
| 主色 | `#9B87C4` | 薰衣草紫，用于主按钮、Logo 背景、重点图标 |
| 主色深 | `#7B6BA3` | 主色按下/强调时可选用 |
| 主文案 | `#1A1A1A` | 标题、重要说明 |
| 次要文案 | `#666666` | 副标题、说明、链接文字 |
| 占位符 | `#999999` | 输入框 placeholder |
| 边框 | `#E0E0E0` | 输入框、次要按钮描边 |
| 错误 | `#C62828` | 错误提示、必填提示 |
| 成功 | `#5C9B5C` | 成功状态（如创建成功图标背景） |
| 卡片 | `#F8F8F8` | 列表项、设置区块背景 |

---

## 2. 圆角

- **输入框**：`12px`（borderRadius.input）
- **主/次按钮**：`12px`（borderRadius.button）
- **Logo 圆形**：50% 或固定数值（如 44、36）形成正圆

---

## 3. 间距

建议使用统一间距阶梯（见 `apps/mobile/src/theme.ts`）：

- `xs`: 8
- `sm`: 12
- `md`: 16
- `lg`: 24
- `xl`: 32

同一区块内元素间距保持一致；区块之间用 `lg` / `xl` 区分层次。

---

## 4. 字体与字号

- **字体**：系统默认无衬线（sans-serif），不引入额外字体时保持系统风格。
- **标题**：22–24px，`fontWeight: "600"`。
- **副标题 / 说明**：14–15px，颜色用次要文案色。
- **输入框**：16px，避免 iOS 自动放大。
- **主按钮文案**：16px，`fontWeight: "600"`，白色字。
- **链接 / 辅助操作**：15px，次要文案色。

---

## 5. 组件规范

### 5.1 主按钮（Primary Button）

- 背景：主色 `#9B87C4`
- 文案：白色、16px、加粗
- 高度：约 48px（paddingVertical 16）
- 圆角 12px，水平铺满或固定最大宽度
- 禁用：`opacity: 0.7`

### 5.2 次要按钮（Secondary / 描边按钮）

- 背景：透明
- 描边：1.5px，`#E0E0E0`
- 文案：主文案色，16px
- 圆角、高度与主按钮一致

### 5.3 输入框（TextInput）

- 边框：1.5px，`#E0E0E0`
- 圆角：12px
- 内边距：约 14–16px
- placeholder：占位符色 `#999999`
- 正文：主文案色，16px

### 5.4 Logo / 品牌图标

- 圆形背景：主色，正圆（宽高一致，borderRadius 为宽高一半）
- 图标：白色，MaterialIcons 等，尺寸约 40–48
- 欢迎页可略大（如 88px 圆），内页可略小（如 72px 圆）

### 5.5 页面结构

- 使用 `SafeAreaView` 包整页，背景为白。
- 内容区：水平居中，`maxWidth: 320–360`，左右 `paddingHorizontal: 24`。
- 典型顺序：Logo → 标题 → 副标题 → 表单项 → 主按钮 → 辅助链接。

---

## 6. 文案与交互

- 登录：强调「欢迎回来」+ 家庭 Token 输入 +「登录」。
- 注册：强调「创建家庭」+ 家庭名称（选填）+「注册」；成功后明确提示保存 Token。
- 错误提示：放在输入框下方，使用错误色，简短一句说明。
- 链接/辅助操作：放在主按钮下方，居中，不强调下划线。

---

## 7. 代码引用

主题常量集中在 **`apps/mobile/src/theme.ts`**：

- `colors`：上述色值
- `spacing`：xs / sm / md / lg / xl
- `borderRadius`：input / button

新页面或组件应优先从 `theme.ts` 取色和间距，以保持与本文档一致。

---

## 8. 参考界面

- 欢迎页：`WelcomeScreen` — 大 Logo + 标题 + 双按钮（登录 / 注册）。
- 登录页：`LoginScreen` — 小 Logo + 欢迎回来 + Token 输入 + 主按钮 +「去注册」链接。
- 注册页：`RegisterScreen` — 创建家庭表单 + 成功态（绿色勾 + Token 卡片）。

以上三屏可作为后续新增登录/注册相关页面的视觉与布局参考。

---

## 9. 其他已统一风格的页面

- **首页（HomeScreen）**：今日摘要 + 卡片式入口（快速记录、时间线、设置），每项左侧圆形图标、主色点缀、右侧箭头。
- **快速记录（QuickAddScreen）**：标题 + 宝宝/类型为芯片选择（选中主色底），表单使用统一 input 样式，底部主按钮「添加记录」。
- **时间线（TimelineScreen）**：筛选宝宝为芯片；每条记录为卡片，类型用主色徽章 + 时间 + 简要 payload。
- **设置（SettingsScreen）**：分区标题 + 卡片容器，FaceID 开关（主色轨道）、输入框、主按钮/次要按钮。
- **锁屏（LockScreen）**：与登录页一致，圆形主色 Logo + 标题 + 主按钮「解锁」。

以上页面均引用 `theme.ts` 的 `colors`、`spacing`、`borderRadius`，新增页请保持一致。
