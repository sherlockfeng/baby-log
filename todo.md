## 模块二：邀请流程

- [ ] `[研发]` **2.1 邀请 API**
  - [ ] `POST /invites` — 创建邀请（admin only）
  - [ ] `POST /invites/:code/accept` — 接受邀请（无需 auth）
  - [ ] `GET /invites` — 查看未使用的邀请列表（admin only）
  - [ ] `DELETE /invites/:id` — 撤销邀请（admin only）
  - [ ] 6 位邀请码、7 天有效、一码一用

- [ ] `[研发]` **2.2 邀请 UI**
  - [ ] 设置页 → "家庭成员" → "邀请家人"
  - [ ] 角色选择 + 称呼输入（如"奶奶"、"外公"、"李阿姨"）
  - [ ] 生成邀请码 + 可复制的邀请文案
  - [ ] 通过微信/短信/系统分享

- [ ] `[研发]` **2.3 加入家庭流程**
  - [ ] 欢迎页新增"加入家庭"入口
  - [ ] 输入邀请码 → 设置自己的称呼 → 获得独立 token
  - [ ] 邀请码使用后自动失效

- [ ] `[测试]` **2.4 邀请流程 E2E**
  - [ ] 完整邀请流程截图验证（创建→分享→接受→加入成功）
  - [ ] 过期/已用邀请码拒绝验证
